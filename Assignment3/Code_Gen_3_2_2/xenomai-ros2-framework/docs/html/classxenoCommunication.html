◆ sendDouble()

int xenoCommunication::sendDouble ( double  toSend)

Xenomai IPC based send function. Send the supplied double to the port set by the initializer of the class, best used within XDDP. Function is set to MSG_DONTWAIT, therefore if no room in XDDP/IDDP buffer, send will be skipped.
